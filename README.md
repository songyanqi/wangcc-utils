## wangcc-utils
The wangcc-utils library exported as esm and cjs.

### Install:

```bash
$ npm i --save wangcc-utils
```
or

```
$ yarn add wangcc-utils
```

### Usage:
```javascript
// Load the full build.
import util from "wangcc-utils";
// Load some tool function.
import { getPassedTime } from "wangcc-utils";
```

### Keywords

`utils`
